# YoutubeDownloader
Youtube video, audio and playlist downloader with gui and multithreading in Java </br>
# Used software:
- Youtube-dl
- Javafx
- Kotlin
- Kotlinx-Coroutines
- Kotlinx-Coroutines-Javafx
- Maven
- Kotlinx-Serialization
